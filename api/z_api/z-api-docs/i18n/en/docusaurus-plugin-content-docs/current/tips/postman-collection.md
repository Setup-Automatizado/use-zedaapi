## Introduction to the FUNNELCHAT Postman Collection

To facilitate development and integration with our API, we provide a Postman collection that includes all FUNNELCHAT endpoints, parameters, and request examples. This collection helps developers test endpoints quickly and efficiently, allowing them to explore the API's functionalities without the need to create requests manually.

Below is the link to access FUNNELCHAT Postman collection:

[Postman Collection Link](https://www.postman.com/docs-funnelchat/funnelchat-s-public-workspace/collection/gwri249/funnelchat-collection)
